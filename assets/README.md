# Onde Ver Filmes e Séries

Site estático pronto para publicar com GitHub e Vercel.

## Publicar no GitHub

1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta para a raiz do repositório.
3. Confirme que o arquivo `index.html` ficou diretamente na raiz.
4. Faça o commit e envie para a branch principal.

## Publicar na Vercel

1. Acesse a Vercel e escolha `Add New` > `Project`.
2. Importe o repositório do GitHub.
3. Use `Framework Preset: Other`.
4. Deixe `Build Command` vazio.
5. Deixe `Output Directory` vazio ou use `./`.
6. Clique em `Deploy`.
7. Depois do deploy, vá em `Settings` > `Domains`.
8. Adicione o domínio `ondeverfilmeseseries.com`.
9. Configure o DNS do domínio conforme os registros informados pela Vercel.
